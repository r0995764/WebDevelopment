Opdracht 3

  • Inline style attributen?

    1. De inline style attributen worden altijd aangeduid na de tag name binnen het element en begint met style "=" en daarna komt tussen aanhalingstekens wat je wilt dat het element doet.

  • <style>?

    1. De gewone style attributen worden door een element style in de head section aangeduid.

  • <link> verwijzingen naar aparte CSS-bestanden?

    1. Dit wordt gedaan door een link element toe te voegen met een href die verwijst naar een ander bestand en dit zal op het einde eindigen met een.CSS, hierbij is dit dat het CSS bestand,
      en ook een rel wordt gebruikt, dit duidt het soort stijlblad aan. Als dit CSS is, is dit altijd een stylesheet.
      Meestal zal dit naar 1 enkel CSS bestand een verwijzing leggen.
